## __controller/GestorTareas.java__

En este archivo se define lo que se conoce como __l贸gica de negocio__, es decir, las operaciones __CRUD__ (Crear, Leer, Actualizar, Eliminar) sobre una lista de tareas.

Adem谩s, controla el ID autoincremental y mantiene la lista de tareas en memoria.

```java
package controller;

import model.Tarea;
import java.util.ArrayList;

public class GestorTareas {
    private ArrayList<Tarea> tareas = new ArrayList<>();
    private int proximoId = 1;

    public void agregarTarea(String descripcion) {
        tareas.add(new Tarea(proximoId++, descripcion));
    }

    public void eliminarTarea(int id) {
        tareas.removeIf(t -> t.getId() == id);
    }

    public void completarTarea(int id) {
        for (Tarea t : tareas) {
            if (t.getId() == id) {
                t.completar();
                break;
            }
        }
    }

    public ArrayList<Tarea> obtenerTareas() {
        return tareas;
    }
}
```
