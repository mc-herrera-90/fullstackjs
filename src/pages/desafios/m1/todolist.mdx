### <i class="fas fa-file-code" style={{color: 'var(--main-secondary)'}}></i> __controller/GestorTareas.java__

En este archivo se define lo que se conoce como __l√≥gica de negocio__, es decir, las operaciones __CRUD__ (Crear, Leer, Actualizar, Eliminar) sobre una lista de tareas.

Adem√°s, controla el ID autoincremental y mantiene la lista de tareas en memoria.

```java
package controller;

import model.Tarea;
import java.util.ArrayList;

public class GestorTareas {
    private ArrayList<Tarea> tareas = new ArrayList<>();
    private int proximoId = 1;

    public void agregarTarea(String descripcion) {
        tareas.add(new Tarea(proximoId++, descripcion));
    }

    public void eliminarTarea(int id) {
        tareas.removeIf(t -> t.getId() == id);
    }

    public void completarTarea(int id) {
        for (Tarea t : tareas) {
            if (t.getId() == id) {
                t.completar();
                break;
            }
        }
    }

    public ArrayList<Tarea> obtenerTareas() {
        return tareas;
    }
}
```

### <i class="fas fa-file-code" style={{color: 'var(--main-secondary)'}}></i> __view/Menu.java__


Esta clase se encarga de la **interacci√≥n con el usuario**. Muestra un men√∫ en la consola y permite seleccionar las operaciones disponibles.

package view;

import controller.GestorTareas;
import model.Tarea;

import java.util.Scanner;

public class Menu {
    private GestorTareas gestor;
    private Scanner scanner;

    public Menu(GestorTareas gestor) {
        this.gestor = gestor;
        this.scanner = new Scanner(System.in);
    }

    public void mostrar() {
        int opcion;
        do {
            System.out.println("\nüìã MENU TODO LIST:");
            System.out.println("1. Agregar tarea");
            System.out.println("2. Ver tareas");
            System.out.println("3. Completar tarea");
            System.out.println("4. Eliminar tarea");
            System.out.println("0. Salir");
            System.out.print("Elige una opci√≥n: ");
            opcion = scanner.nextInt();
            scanner.nextLine();

            switch (opcion) {
                case 1 -> {
                    System.out.print("Descripci√≥n de la tarea: ");
                    String desc = scanner.nextLine();
                    gestor.agregarTarea(desc);
                }
                case 2 -> {
                    System.out.println("\nTAREAS:");
                    for (Tarea t : gestor.obtenerTareas()) {
                        System.out.println(t);
                    }
                }
                case 3 -> {
                    System.out.print("ID de la tarea a completar: ");
                    int id = scanner.nextInt();
                    gestor.completarTarea(id);
                }
                case 4 -> {
                    System.out.print("ID de la tarea a eliminar: ");
                    int id = scanner.nextInt();
                    gestor.eliminarTarea(id);
                }
                case 0 -> System.out.println("üëã Hasta luego.");
                default -> System.out.println("‚ö†Ô∏è Opci√≥n no v√°lida.");
            }
        } while (opcion != 0);
    }
}
